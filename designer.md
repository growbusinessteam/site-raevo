# Raevo Design System

## Brand Direction

Raevo uses a warm modern luxury direction: precise, quiet, strategic and technological. The brand should feel like intelligence made visible, with enough warmth to stay human and enough discipline to feel executive.

Core ideas:

- Visao para evoluir.
- Sistemas para crescer.
- Estrategia, inteligencia, transformacao and sustainable growth.
- Premium B2B technology for companies in transformation.

## Color Tokens

| Token | Hex | Use |
| --- | --- | --- |
| Pedra | `#E9E4DA` | Main mineral background, calm surfaces |
| Areia | `#DCCFBE` | Secondary warm background, panels |
| Taupe | `#B8AB98` | Borders, muted text, dividers |
| Carvao | `#1F1F1F` | Primary text, dark sections |
| Champagne | `#C7A97A` | Premium CTA, icons, fine rules |
| Cian | `#00B8C6` | Transformation accent, data highlights |

## Typography

- Display and section titles: Sora.
- Body, navigation, labels and forms: Inter.
- Headlines should be spacious, geometric and calm.
- Body copy should stay practical, direct and easy to scan.

## Visual Rules

- Use light mineral fields and dark charcoal fields as the main contrast.
- Keep lines thin, dividers intentional and spacing generous.
- Use little border radius: 4px to 8px maximum for cards and buttons.
- Use Cian sparingly, as a signal of transformation or active intelligence.
- Use Champagne for primary CTAs, icon lines and premium emphasis.
- Avoid generic gradients, decorative blobs and loud animation.
- Use subtle paper/concrete texture through CSS noise and brand images.
- Keep page sections editorial and full-width; use cards only for repeated items.
